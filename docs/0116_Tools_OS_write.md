# write
- 发送消息给指定用户终端

```
write user [tty]
```


## UseCase

```
$ write user_1  pts/0
```
> 以上命令执行之后可以发送消息，结束 使用 <kbd>Ctrl</kbd> + <kbd>C</kbd>
