# shuf 随机选取工具

## Brief

shuf 命令可以随机提取一个文本或者输入的不同行，输出到文件或者标准输出中。

- 可以用来随机选取文件

## UseCase

- **随机选取三张图片**

```
ls images|shuf -n 3
```
