# VNC System

  - VNC - Virtual Network Computing/Virtual Network Console - 图形桌面共享系统
  - VNC 由AT\&T 的剑桥研究实验室开发，可实现远程图像显示和控制。
  - RFB protocol - `remote framebuffer protocol`

## RFB Protocol

  - 1998 V3.3
  - 2003 V3.7
  - 2007 v3.8
  - 2011 IETF RFC
